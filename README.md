# Harsh Padher — Portfolio

A clean, responsive personal portfolio website built with pure **HTML**, **CSS**, and **JavaScript** — no frameworks, no build step, no dependencies.

## Live Preview

Open `index.html` directly in any modern browser. No server required.

## Project Structure

```
harsh-portfolio/
├── index.html      # Semantic HTML markup
├── style.css       # All styles & responsive layout
├── script.js       # Dynamic content & interactivity
├── assets/         # Images, icons, or other static files
├── README.md
├── LICENSE
└── .gitignore
```

## Features

- **Responsive** — mobile-first layout with hamburger nav drawer
- **Smooth scrolling** — active section tracking in the navbar
- **Animated nav** — frosted-glass effect on scroll
- **Zero dependencies** — runs straight from the file system or any static host
- **GitHub Pages ready** — push and enable Pages; it just works

## Sections

| Section | Description |
|---------|-------------|
| Hero | Name, tagline, and social links |
| About | Bio and personal strengths |
| Skills | Programming, web, database, and tooling skills |
| Projects | Portfolio of notable work |
| Education | Academic background and training |
| Contact | Phone, email, and social links |

## Deploy to GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Set source to `main` branch, `/ (root)`.
4. Your site will be live at `https://<username>.github.io/<repo>/`.
